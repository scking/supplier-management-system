# 供应商管理系统本地部署说明

## 一、目录

项目根目录：

- `/Users/songchao/Documents/supplier-management-system`

## 二、启动方式

优先使用 Docker Compose 本地启动：

```bash
cd /Users/songchao/Documents/supplier-management-system
docker compose up -d --build
```

或使用脚本：

```bash
./scripts/start-local.sh
```

## 三、访问地址

- 前端：`http://127.0.0.1:3180`
- 后端：`http://127.0.0.1:9180/api`
- MySQL：`127.0.0.1:33306`

## 四、说明

- 首次启动会自动初始化数据库表和种子数据
- 当前登录依赖统一门户 SSO
- 若本机未启动统一门户，则 SSO 登录页可打开，但无法完成真实登录换票
- 当前机器若未安装 Java，也可通过 Docker 启动后端
